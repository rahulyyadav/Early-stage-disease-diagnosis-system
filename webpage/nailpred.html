OUR PROJECT ON NAIL DISEASE

Human nail can be used for the prediction of various systemic and dermatological diseases. The proposed system – Nail Image Processing System – helps us analyze human nails and thereby assist in predicting various nail diseases. The input to the proposed system is a Human Palm Image. The nail portion is segmented, and nail color, shape, and texture features are extracted and taken together to form a feature vector. This is then analyzed for the diagnosis of various nail diseases. This system can help doctors in early diagnosis and treatment.

MODELS EMPLOYED FOR THE PROJECT

Empowering detection with powerful Transfer Learning Models.

VGG16

VGG-16 is a convolutional neural network that is 16 layers deep, capable of classifying images into 1000 categories.

RESNET50

ResNet50 is a powerful 50-layer CNN, adept at classifying images into 1000 object categories with high accuracy.

INCEPTIONV3

InceptionV3 is a sophisticated 48-layer CNN widely utilized for precise image classification into 1000 object categories.

XCEPTION

Xception is a cutting-edge 71-layer CNN optimized for advanced image classification tasks across numerous categories.
